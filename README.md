Criei uma lista de nomes.
Usei a expressão Lambda para filtrar e mostrar apenas os nomes que começam com a letra "A".
Utilizando uma interface funcional própria chamada FiltroNome que tenha um método abstrato "boolean testar(String nome){}".
Criei a interface FiltroNome com a anotação @FunctionalInterface e instancie ela no método main.
Utilizei o método stream().filter() com uma expressão Lambda para aplicar o filtro para exibir os nomes filtrados no console.
